征服键盘尚未成功

发表于 2023-04-16

打了那么多年 OI，这人居然不会盲打！小学教指法，这人却不好好学！

野路子敲键盘也不是一指禅，其实挺快的，但时不时低头影响思路。我从省选回来开始练习盲打，现在过去了两个星期，基本可以不看键盘写下本文。只是经常输错，速度比坚持多年的野路子慢得多。

搜了一些学打字的网站，无非从 f、j 出发，随机（或者按我看不懂的设计？）字符让你跟着打，然后扩大随机范围。我也写了一个。

（更新：推荐网站 keybr.com，它让你打的字符串大多是有意义的单词，还可以专练编程语言。）

重新认识字母并用手打出它们很有成就感。刚开始节奏很重要，读一个字母打一个字母，像弹琴，虽然我没学过弹琴。我开始希望打字可以和朗读一样“不用脑子”，可是随机字符集达到 sdfjkl 时错误率就居高不下。我发现敲随机字符经常忘记键位，毕竟不如朗读熟练。改成敲代码就不容易忘，敲拼音更简单，因为拼音字母组合很有限。

后来就不敲随机字符了，改敲《六国论》，舒服多了。

左手中指敲 c 太别扭，用食指敲多好。不知道为啥网上流传的指法图都把左上-右下倾斜的一列归给一根手指。左手明明右上-左下倾斜才舒服。

敲代码的问题是括号（和引号）。以前我手是随便摆的，输入左边括号自动匹配右边，要从括号出来得按右方向键。现在手放在固定位置，按方向键就有点麻烦。但如果没有括号匹配，右手小拇指工作量几乎翻倍。我觉得设计键盘的时候应该从右侧按键中分一列给左侧。
